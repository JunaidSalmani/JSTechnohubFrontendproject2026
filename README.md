## 🏗️ Technical Architecture

The platform is built with a modern, scalable directory structure:

```text
├── public/              # Static assets (Favicons, manifest, robots.txt)
├── build/               # Production-ready optimized code
├── src/
│   ├── assets/          # Global media (Images, brand logos, icons)
│   ├── components/      # Reusable UI modules (CourseCards, Navbar, Footer)
│   ├── service/         # API integrations and business logic handlers
│   ├── app.js           # Core routing and application logic
│   ├── app.css          # Main layout and global styling
│   ├── index.js         # Entry point for React DOM
│   └── index.css        # Tailwind/Base CSS resets
└── README.md            # Comprehensive project documentation
```

## 🤝 How to Contribute

We welcome contributions! Whether it's fixing a bug, adding a new course module, or improving the UI, follow these steps:

### 1. Clone the Repository

Open your terminal and run:

```bash
git clone "https://github.com/JunaidSalmani/JSTechnohubFrontendproject2026.git"
```

### 2. Create a Feature Branch

```
git checkout -b feature/AmazingFeature
```

### 3. Commit and Push

Follow professional commit naming conventions (e.g., feat:, fix:, docs:):

```
git add .
git commit -m "feat: add interactive course cards"
git push origin feature/AmazingFeature
```
